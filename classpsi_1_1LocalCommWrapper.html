Performs element-by-element sum of all data from all nodes. The sum will either appear in a new buffer or will overwrite the original data.

Parameters
dataThe array of data to be summed. If no receive buffer is given, this is overwritten with summed data.
nThe size of the array. This is the number of elements, not the number of bytes.
receive_bufferOptional receive buffer. If given, summed data is placed in this array. If used, this must be allocated before entering method.
targetData is summed on target. By default all nodes receive data.
