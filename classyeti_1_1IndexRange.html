IndexRange::IndexRange ( uli  start,
const SubindexTuplePtr &  tuple,
uli  nper 
)

Builds an index range with sliced subranges. Based on nper, a set of subranges are sliced from tuple to create a subindex. For example, if tuple had 8 elements and nper was 3, this would create an index range with 3 subindices. Subindex 0 would contain tuple[0]->tuple[2] as subindices, subindex 1 would contain tuple[3]->tuple[5], subindex 2 would have tuple[6]->tuple[7].

Parameters
startThe starting index for the index range
tupleThe set of index ranges to build from
descr
nperThe desired number of indices per subindex
