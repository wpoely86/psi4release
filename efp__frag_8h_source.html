26 /* This class will define a fragment for use in situations where a fragment
27 is defined by space-fixed coordinates: 3 center-of-mass and 3 Euler angle
28 coordinates.
29 
30 This class is intended to be used for fixed-body EFP optimizations in QChem.
31 In QChem, the gradient is already computed in these space-fixed coordinates.
32 Thus, it is not necessary to write the code for the B-matrix to do an
33 optimization. Eventually, a B-matrix will be necessary in order to, for
34 example, transform a cartesian analytic Hessian into internal coordinates for
35 these optimizations.
36 
37 Since the angular coodinates are be described as linear combinations of positions of
38 atoms, a new type of B-matrix will be necessary. We might be able to use
39 the principle axes to define the coordinates (derived by WDA), but we would
40 still have to relate these to the space-fixed coordinates of the gradient
41 computed by QChem.
42 */
