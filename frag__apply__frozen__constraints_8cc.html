Detailed Description

apply constraints specified by given strings

FRAG::apply_frozen_constraints(string) parses string for the atoms that define bonds, bends, and angles to be frozen. Return false if there are none.

FRAG::apply_fixed_constraints(string) parses string for the atoms that define bonds, bends, and angles to be fixed. Return false if there are none.
