chkpt_set_mo_phases(): Set the phase of the MO's according to the standard that the first element in every column is always positive. This may help keep the phase consistent for more complicated problems like natural orbitals.

Parameters
coeff= MO coefficient matrix
nrows= number of rows in MO coefficient matrix
ncols= number of columns in MO coefficient matrix

Note: since it only looks at the first element in each column, it will not work for matrices where that element can be zero by symmetry. So, this function is only helpful when called for an irrep block at a time.

David Sherrill, July 2002
