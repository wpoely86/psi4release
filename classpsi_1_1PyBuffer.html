Detailed Description

template<class T>
class psi::PyBuffer< T >

A thin wrapper to the NumPy array interface.

PyBuffer instances can be used to expose the pointer of a given C++ buffer to the NumPy array interface without copying any data. For more details, see http://docs.scipy.org/doc/numpy/reference/arrays.interface.html
