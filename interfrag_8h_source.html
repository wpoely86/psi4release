35 /*
36 INTERFRAG is a set of coordinates between two fragments (A and B).
37 Up to 3 reference atoms on each fragment (dA[3] and dB[3]) are defined
38 as either
39  If INTERFRAGMENT_MODE == FIXED,
40  linear combinations of the atoms in A and B, using A_weight and B_weight,
41  or if INTERFRAGMENT_MODE == PRINCIPAL_AXES
42  1. the center of mass
43  2. a point a unit distance along the principal axis corresponding to the largest moment.
44  3. a point a unit distance along the principal axis corresponding to the 2nd largest moment.
45 
46 For now, the six coordinates formed from the d_K^{A,B} sets are assumed to be the
47 following in this canonical order:
48 D[0], RAB = distance between dA[0] and dB[0]
49 D[1], theta_A = angle, dA[1]-dA[0]-dB[0]
50 D[2], theta_B = angle, dA[0]-dB[0]-dB[1]
51 D[3], tau = dihedral angle, dA[1]-dA[0]-dB[0]-dB[1]
52 D[4], phi_A = dihedral angle, dA[2]-dA[1]-dA[0]-dB[0]
53 D[5], phi_B = dihedral angle, dA[0]-dB[0]-dB[1]-dB[2]
54 
55 inter_geom[0] = dA[2]; // For simplicity, we sort the atoms in the 'inter_geom'
56 inter_geom[1] = dA[1]; // according to the assumed connectivity of the coordinates.
57 inter_geom[2] = dA[0]; // For A, this reverses the order in which weights are
58 inter_geom[3] = dB[0]; // provided to the constructor.
59 inter_geom[4] = dB[1];
60 inter_geom[5] = dB[2];
61 */
