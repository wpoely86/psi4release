Detailed Description

template<class T>
class smartptr::Set< T >

Container that automatically converts types. The vector constructor does not allow automatic conversion of dynamic types, e.g. A vector of SimpleInternalCoordinates cannot be directly converted to a vector of InternalCoordinates. The implicit template constructor of Set automatically performs type conversions.
