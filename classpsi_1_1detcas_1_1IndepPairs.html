set()

This function sets the independent pairs info

Parameters
nirr= number of irreps in point group
num_ras= number of RAS spaces
ras_opi= number of orbitals per irrep per RAS space
ras_orbs= ras_orbs[ras][irr][cnt] gives an orbital number
fzc= number of frozen core orbs per irrep
fzc_orbs= fzc_orbs[irr][cnt] gives an orbital number
cor= number of restricted core orbs per irrep
cor_orbs= cor_orbs[irr][cnt] gives an orbital number
vir= number of restricted virtual orbitals per irrep
vir_orbs= vir_orbs[irr][cnt] gives an orbital number
fzv= number of frozen virtual orbitals per irrep
fzv_orbs= fzv_orbs[irr][cnt] gives an orbital number
ci2relpitz= maps CI orbitals to relative Pitzer indices
ignore_ras_ras= ignore RAS/RAS independent pairs
ignore_fz= ignore FZC and FZV from all independent pairs
