nlayers_extraAt minimum three metadata layers are required. The ''node'' layer creates a grid for distribution of tensor blocks across nodes. The ''thread'' layer distributes tasks for threads within a given node. The final layer or ''day'' layer generates a grid for each individual data block. Extra metadata layers can be inserted between the bottom ''data'' layer and the ''thread'' layer.
