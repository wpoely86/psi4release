35 /*
36 
37  Constrained Unrestricted Hartree-Fock
38 
39  Reference: T. Tsuchimochi and G.E. Scuseria, J. Chem. Phys. 133,
40  141102 (2010)
41 
42  This is an alternative formulation of ROHF as a contrained UHF. A
43  Lagrangian constraint is placed on the usual UHF procedure to remove
44  the spin contamination. The result is an ROHF energy and semicanonical
45  ROHF orbitals. The need to pick coupling coefficients is removed.
46  Koopmans' theorem is valid for CUHF orbital energes.
47 
48  It is claimed that CUHF does not suffer from the convergence problems
49  of certain ROHF implementations (not sure how PSI's ROHF code does).
50  CUHF retains the UHF-like trait that Ca != Cb. Also, the converged CUHF
51  wavefunction yields the correct value for <S^2>, however, this is only
52  true at convergence. It is possible that this increased flexibility
53  improves convergence.
54 
55  -- EGH, August 15th, 2011
56 
57  TODO:
58 
59  Probably can't handle NSO != NMO right now, should either fix this code
60  or the transform functions from Matrix.
61 
62  Using the UHF form for the Lagrangian, this is probably correct, but
63  should be checked.
64 
65 */
