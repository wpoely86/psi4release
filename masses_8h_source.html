23 /*
24 ** The new PSI standard header file for atomic masses.
25 **
26 ** Created by Matt Leininger, July 1995
27 **
28 ** This file contains three arrays. The first is a list of atomic masses
29 ** for a number of isotopes. The second is a list of isotope labels which
30 ** correspond to the masses in the first array. The mass labels given
31 ** are the same as those in W. D. Allen's INTDER95. The final array is a
32 ** list of atomic masses for the first several elements, with the most
33 ** common isotopomer mass given. This last array is most useful for
34 ** converting an atomic number into an atomic mass.
35 **
36 ** Please be sure that any modifications to this file are reflected in
37 ** _all three_ arrays if necessary.
38 **
39 ** Modifications:
40 ** 10/08/99 EFV - Changed an2masses[0] to 0.0000
41 ** (ghost atom's weight should be 0)
42 **
43 ** 08/16/02 EFV - Updated atomic_masses[] and an2masses for H-Ar using NIST database
44 ** of Atomic Weights and Isotopic Compositions (www.nist.gov)
45 **
46 ** 08/23/02 EFV - Use lithium isotope 7 as the default (most abundant)
47 **
48 ** 03/03/08 MT - extended set of atoms by data from mathematica script. For an2masses
49 ** for naturally occurring atoms, the most abundant is used, otherwise the
50 ** atom with the longest halflife time.
51 **
52 */
