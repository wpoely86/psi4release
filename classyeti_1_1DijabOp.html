void DijabOp::element_op ( uli  nblock,
const uli *  index_starts,
const uli *  sizes,
double *  data 
)
virtual

Carries out the operation characteristic of the DiaOp class. Specifically, it multiplies each element of the block from data by 1/(ci*ei + cj*ej + ca*ea + cb*eb). (Note that this should never be called at the user level).

Parameters
nblockThe number of items in the block being operated on. (Unused in current implementation).
index_startsPointer to a four element array containing the starting indices for i, j, a, and b in that order.
sizesPointer to a four element array containing the number of i, j, a, and b elements in the block to be operated on.
dataPointer to the data block being operated on.
