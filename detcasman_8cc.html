Detailed Description

Determinant CASSCF/MCSCF manager.

Program to manage the iteration of (transqt, detci, clag, detcas) required for orbital optimization using the DETCAS program.

This program does not really do much...it simply iterates until convergence or until iterations are exhausted. It would not be necessary if it were possible to rewrite the PSI driver to be more general and allow non-crashing exits out of loops; somebody who knows how to do this should do it.

C. David Sherrill University of California, Berkeley April 1998

Modification History:

  • Modified 10 February 1999 by C. David Sherrill - Added the ability to parse the orbital optimization log file (file14) so this information can be used to allow looser convergence on the CI during early iterations.
